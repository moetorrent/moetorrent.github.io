### Features & Roadmap

#### Currently Implemented:

- Polished and cute user interface
- No ads and 100% open-source
- Native multi-window desktop application (via Tauri)
- Bundled, auto-starting Transmission daemon
- Magnet URI and `.torrent` file parsing
  - Displays creation date, size, and Info Hashes (v1 and v2)
  - Checks target drive free disk space
- Interactive file browser inside torrent add window
  - Collapsible file/directory tree view
  - Real-time size calculation of selected files
- Torrent management and queue operations (start, pause, stop, delete)
- Real-time transfer statistics and diagnostics
  - Global upload and download speed meters
  - Active seed/peer counts and ETA per torrent

#### Planned:

- Support for selective file downloads
- Advanced configurations (speed limits, ports, default download path)
- Customizable themes and mascot artwork options
- Expanded diagnostics (peer clients, tracker list, speed graphs)
- Desktop integration (system tray, background running, completion notifications)
- Sequential downloading support (for media streaming)
